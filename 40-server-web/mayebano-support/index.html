下载脚本

1. 执行前确认
  1. 1不是应用商店上传安装，也不是固件升级包。
  2. 2先在系统安全页开启 SSH，保存并应用。

SSH 安全页

在 NRadio 后台打开系统安全页，启用 SSH，保存并应用。

http://192.168.66.1/cgi-bin/luci/admin/system/security 一键打开 SSH 安全页
2. 准备 SSH 工具
  • MobaXterm使用 SSH 终端登录路由器后粘贴命令。
  • FinalShell / Xshell使用 SSH 终端连接 root 后执行下载命令。
  • PuTTY登录路由器终端后粘贴下面三行命令。
  • 路由器 Web SSH设备已有 Web SSH 时，可直接粘贴终端下载命令。
3. 路由器在线下载

SSH 登录路由器后，在终端里粘贴下面命令，脚本会下载到 /root 并立即执行。

  1. SSH 新建连接：主机 192.168.66.1，用户 root，密码填路由器 root 密码。
  2. 进入终端后先确认是在路由器命令行，不是在浏览器地址栏或 SSH 软件的本地终端。
  3. 一次性复制下面三行命令，粘贴到 SSH 终端后回车执行。
  4. 看到 NRadio 脚本菜单后，表示下载和启动完成，再按菜单编号继续。
cd /root wget -O ssh-nradio-plugin-installer.sh https://nradio.mayebano.shop/ssh-nradio-plugin-installer.sh sh ssh-nradio-plugin-installer.sh
4. 执行后确认
  • 出现菜单终端出现 NRadio 脚本菜单后，再按编号继续。
  • 按菜单执行安装、美化、卸载都按脚本提示确认。
  • 回后台核对完成后回应用商店确认入口、图标和状态。
5. 不要做
  • 不要把脚本当应用商店安装包上传，也不要当固件升级包。
  • 脚本适用于 NRadio_C8-688 NRadio_C8-668 C5800-688 NBCPE C2000MAX 官方 NROS2.0 系统；标准 OpenWrt 不适用。
  • 执行终端下载命令前，先确认路由器可以访问公网。
  • 不要在未确认 IP 和机型的设备上执行；不同路由器、旁路由和服务器不要混用。
  • 安装和美化入口只按菜单提示执行；终端正在执行时不要直接关闭窗口。
