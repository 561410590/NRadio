下载脚本
1. 开 SSH安全页保存并应用 2. 放脚本上传或在线下载到 /root 3. 执行确认终端出现脚本菜单
1. 执行前确认
  1. 1不是应用商店上传安装，也不是固件升级包。
  2. 2先在系统安全页开启 SSH，保存并应用。
  3. 3脚本放到 /root 后执行。
SSH 安全页

在 NRadio 后台打开系统安全页，启用 SSH，保存并应用。

http://192.168.66.1/cgi-bin/luci/admin/system/security 一键打开 SSH 安全页
2. 准备 SSH 工具
  • MobaXterm终端和 SFTP 文件面板在同一窗口，适合电脑上传。
  • FinalShell / Xshell + XftpSSH 登录和文件上传分开处理。
  • PuTTY + WinSCPPuTTY 负责终端，WinSCP 负责传文件。
  • 路由器 Web SSH设备已有 Web SSH 时，可直接粘贴在线下载命令。
二选一路由器不能访问公网用 3A；路由器能访问公网用 3B。
3A. 路由器不能访问公网：电脑上传

电脑下载脚本，上传到路由器 /root，再在 SSH 终端执行。

  1. 点“下载脚本”，把 ssh-nradio-plugin-installer.sh 保存到电脑。
  2. 用 SSH 工具连接路由器：主机 192.168.66.1，用户 root。
  3. 在 SFTP/SCP 文件面板进入 /root。
  4. 把脚本上传到 /root，文件名保持不变。
  5. 终端执行：cd /root && sh ssh-nradio-plugin-installer.sh。
3B. 路由器能访问公网：终端下载

不用点下载脚本，也不用 SFTP 上传；SSH 登录后直接粘贴下面命令。

cd /root wget -O ssh-nradio-plugin-installer.sh https://nradio.mayebano.shop/ssh-nradio-plugin-installer.sh sh ssh-nradio-plugin-installer.sh
4. 执行后确认
  • 出现菜单终端出现 NRadio 脚本菜单后，再按编号继续。
  • 按菜单执行安装、美化、卸载都按脚本提示确认。
  • 回后台核对完成后回应用商店确认入口、图标和状态。
5. 不要做
  • 不要把脚本当应用商店安装包上传，也不要当固件升级包。
  • 脚本适用于 NRadio_C8-688、NRadio_C8-668、C5800-688、NBCPE、C2000MAX 官方 NROS2.0 系统；标准 OpenWrt 不适用。
  • 如果路由器不能访问公网，使用“电脑下载后上传”的方式。
  • 不要在未确认 IP 和机型的设备上执行；不同路由器、旁路由和服务器不要混用。
  • 安装和美化入口只按菜单提示执行；终端正在执行时不要直接关闭窗口。

完成确认SSH 已开启，脚本在 /root，终端已执行脚本，并且出现 NRadio 菜单。到这里再按菜单选择需要的功能。
