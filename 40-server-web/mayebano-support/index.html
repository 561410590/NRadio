  3. 3通过 SSH 运行总脚本，再按菜单选择功能。

完整安装步骤
  • 先在 NRadio 系统安全页面打开 SSH：/cgi-bin/luci/admin/system/security。
  • 使用 SSH 登录路由器，确认当前设备是带 NRadio 应用商店的官方 NROS 固件。
  • 优先直接在路由器下载脚本；如果路由器不能访问公网，就先下载到电脑，再用 SCP 上传到 /root/。
  • 进入 /root 后执行脚本，阅读首页免责声明，10 秒倒计时结束后按提示同意，再进入菜单；V2 测试脚本会先进入功能分类。

注意事项
  • 脚本适用于 NRadio_C8-688、C5800-688、NBCPE、C2000MAX 官方 NROS 系统；标准 OpenWrt 不适用。
  • 不要在未确认 IP 和机型的设备上执行；不同路由器、旁路由和服务器不要混用。
  • 安装和美化入口只按菜单提示执行；中途关闭终端可能导致插件未完整接入应用商店。

V1.50.5
OpenVPN Mk2 页面美化回写总脚本

把现场验证过的 OpenVPN Mk2 连接中枢、样式层、标准入口、基础配置、高级配置、文件编辑和实例导入入口回写进总脚本；本次只动页面模板与 CSS，不改下载、安装、卸载和运行配置链。

V1.60.0
新增 15 号应用商店美化功能
